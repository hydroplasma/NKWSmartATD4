# ระบบเช็คและติดตามนักเรียนเข้าร่วมกิจกรรมหน้าเสาธง - การวิเคราะห์

## ข้อมูลจากไฟล์ HTML

### ชื่อระบบ
- ระบบเช็คและติดตามนักเรียนเข้าร่วมกิจกรรมหน้าเสาธง
- โรงเรียนน้ำคำวิทยา

### ฟีเจอร์หลัก
1. **ฟอร์มเช็คชื่อนักเรียน** (Attendance Form)
   - เลือกวันที่
   - เลือกช่วงเวลา (เช้า/เที่ยง)
   - ตารางแสดงห้องเรียน
   - สถานะการเช็คชื่อ (มา, ขาด, สาย, ลา, ป่วย)

2. **สรุปตามห้องเรียน** (Classroom Summary)
   - สรุปสถิติต่อห้องเรียน

3. **สรุปภาพรวมทั้งโรงเรียน** (Overall Summary)
   - สรุปสถิติทั้งโรงเรียน

4. **ระบบวิเคราะห์ผู้เรียนรายบุคคล** (Student Analysis)
   - ลิงก์ไปยังระบบวิเคราะห์แยกต่างหาก

### ข้อมูลห้องเรียน
- ม.1 (17 นักเรียน)
- ม.2 (16 นักเรียน)
- ม.3 (23 นักเรียน)
- ม.4 (13 นักเรียน)
- ม.5 (12 นักเรียน)
- ม.6 (20 นักเรียน)

### สถานะการเข้าร่วมกิจกรรม
- มา (Present) - สีเขียว
- ขาด (Absent) - สีแดง
- สาย (Late) - สีเหลือง
- ลา (Leave) - สีฟ้า
- ป่วย (Sick) - สีม่วง

### ธีมสี (ปัจจุบัน)
- ส่วนหัว: สีน้ำเงิน (Indigo-600)
- ปุ่ม: สีน้ำเงิน (Indigo-600)
- พื้นหลัง: สีขาว

## ข้อมูลจากไฟล์ Excel

### โครงสร้างฐานข้อมูล

#### 1. Attendance (399 rows)
- date: วันที่เช็คชื่อ
- period: ช่วงเวลา (morning/noon)
- room_id: รหัสห้องเรียน (m1-1, m2-1, m3-1, m4-1, m5-1, m6-1)
- teacher: รหัสครู
- students: JSON array ของนักเรียน
  - student_id: รหัสนักเรียน
  - status: สถานะ (มา, ขาด, สาย, ลา, ป่วย)
  - reason: เหตุผล
- updated_at: เวลาอัปเดต

#### 2. Students (103 rows)
- student_id: รหัสนักเรียน
- name: ชื่อนักเรียน
- classroom: ห้องเรียน
- number: เลขที่
- status: สถานะ

#### 3. Classrooms (6 rows)
- room_id: รหัสห้องเรียน
- name: ชื่อห้องเรียน
- total_students: จำนวนนักเรียน

#### 4. Periods (3 rows)
- period_id: รหัสช่วงเวลา
- name: ชื่อช่วงเวลา
- time: เวลา

#### 5. Teachers (47 rows)
- teacher_id: รหัสครู
- name: ชื่อครู
- email: อีเมล
- phone: เบอร์โทร
- role: บทบาท
- classroom: ห้องเรียนที่สอน

#### 6. StatusList (5 rows)
- status_id: รหัสสถานะ
- name: ชื่อสถานะ
- color: สีของสถานะ
- description: คำอธิบาย

#### 7. Config (9 rows)
- key: คีย์การตั้งค่า
- value: ค่าการตั้งค่า

## ความต้องการของแอพมือถือ

### ธีมสี
- สีขาว (White)
- สีส้ม (Orange)
- สีดำ (Black)

### ฟีเจอร์ที่ต้องสร้าง
1. หน้าเข้าสู่ระบบ (Login)
2. หน้าแรก (Dashboard) - สรุปสถิติ
3. หน้าเช็คชื่อนักเรียน (Attendance)
4. หน้าสรุปตามห้องเรียน (Classroom Summary)
5. หน้าสรุปภาพรวม (Overall Summary)
6. ระบบจัดการข้อมูล (Admin)

### เทคโนโลยี
- React Native + Expo (Mobile App)
- TypeScript
- TailwindCSS (สำหรับ styling)
- Drizzle ORM (Database)
- MySQL/TiDB (Database)
- Backend API (Node.js/Express)
